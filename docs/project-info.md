## Project Description

Dappazon is a decentralized e-commerce application that demonstrates the capabilities of blockchain technology in creating a transparent and secure online marketplace. Unlike traditional e-commerce platforms where all data is stored on centralized servers, Dappazon stores all product information, transactions, and order history on the Ethereum blockchain.

### Key Learning Outcomes

1. **Smart Contract Development**: Writing secure and efficient Solidity smart contracts
2. **Blockchain Integration**: Connecting frontend applications with blockchain networks
3. **Web3 Development**: Understanding Web3.js and ethers.js libraries
4. **Decentralized Architecture**: Learning about decentralized data storage and management

---

## Technical Implementation

### Smart Contract Features

1. **Product Listing**
   - Store product information on blockchain
   - Categories: Electronics, Clothing, Toys
   - Price in ETH (wei format)

2. **Purchase System**
   - Direct ETH transfers
   - Stock management
   - Purchase confirmation via events

3. **Order Tracking**
   - Order history on blockchain
   - Timestamps for each purchase
   - Buyer address tracking

### Frontend Features

1. **React Application**
   - Component-based architecture
   - State management with React hooks
   - Responsive design

2. **Blockchain Connection**
   - MetaMask integration
   - Ethers.js for contract interaction
   - Real-time data fetching

3. **Demo Mode**
   - Works without blockchain connection
   - Sample data for testing
   - Full UI functionality
